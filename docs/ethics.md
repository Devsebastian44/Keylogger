# Política de Ética y Uso Responsable

Este documento establece las directrices bajo las cuales se debe utilizar este framework de investigación.

## 1. Propósito Educativo
Este proyecto existe para ayudar a estudiantes y profesionales a entender cómo operan las amenazas reales para poder defenderse mejor de ellas.

## 2. Consentimiento Explícito
**NUNCA** ejecutes este software en una computadora o red que no sea de tu propiedad o para la cual no tengas un permiso por escrito explícito.

## 3. Límites Legales
El desarrollo, posesión y uso de "malware" o herramientas de monitoreo están regulados por leyes locales e internacionales (ej. Computer Fraud and Abuse Act en EE.UU., Ley de Delitos Informáticos en varios países de LATAM). El incumplimiento de estas leyes puede resultar en cargos criminales graves.

## 4. No Proliferación
No compiles ni distribuyas binarios ejecutables de este código a personas que no comprendan su naturaleza técnica y ética.

---
**Recuerda:** La verdadera habilidad de un profesional de ciberseguridad no es saber romper, sino saber proteger.
